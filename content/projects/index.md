## Deepfake Detector

Hybrid system for detecting AI-generated images, combining spectral analysis (FFT), classical ML (Random Forest), and deep learning (Xception CNN with Transfer Learning).

Trained on 100k images from the "140k Real and Fake Faces" dataset using Google Colab (Tesla T4). Best result: **75.2% validation accuracy**, AUC **0.829** after 6 epochs. Includes an Explainable AI module powered by GPT-4o for interpreting spectral anomalies.

Built as part of my academic research on information coding and storage theory.

**Tech:** Python, TensorFlow, Streamlit, OpenAI API
**Repo:** [github.com/vataseradu/deepfake-detector](https://github.com/vataseradu/deepfake-detector)
**Live demo:** [deepfake-detector on Streamlit](https://deepfake-detector-tcsivtstcsitcsi.streamlit.app/)
