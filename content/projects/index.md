## Deepfake Detector

Hybrid system for detecting AI-generated images, combining spectral analysis (FFT), classical ML (Random Forest), and deep learning (Xception CNN with Transfer Learning). Best result: **75.2% accuracy**, AUC **0.829**.

**Tech:** Python, TensorFlow, Streamlit, OpenAI API
